本文档是用Electron打包。
需要安装依赖，200mb 左右，然后下载依赖失败，请使用淘宝镜像
# 设置 Electron 镜像为淘宝源
export ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/"
第一步，参加package.json文件
"name": "writing-assistant",
  "version": "1.0.0",
  "description": "写作帮手 - 免费的写作软件",
  "main": "main.js",
  "scripts": {
    "start": "electron .",
    "build": "electron-builder",
    "build-mac": "electron-builder --mac",
    "build-mac-arm64": "electron-builder --mac --arm64",
    "build-mac-x64": "electron-builder --mac --x64",
    "build-win": "electron-builder --win"
  },
  "author": "麻昌生",
  "license": "MIT",
  "devDependencies": {
    "electron": "^28.0.0",
    "electron-builder": "^24.6.4"
  },
  "build": {
    "appId": "com.writingassistant.app",
    "productName": "写作帮手",
    "directories": {
      "output": "dist"
    },
    "mac": {
      "target": ["dmg", "zip"],
      "icon": "icon.icns",
      "category": "public.app-category.productivity",
      "hardenedRuntime": true,
      "gatekeeperAssess": false
    },
    "win": {
      "target": "nsis",
      "icon": "icon.ico"
    },
    "files": [
      "**/*",
      "!node_modules/.cache",
      "!dist",
      "!icon.ico"
    ]
  }
}
第二步，准备图标，根目录已经准备了，mac是icon.icns，Windows 是ico.icns
第三步，安装 electron-builder（如果还没安装）
npm install electron-builder --save-dev
第四步，打包架构；
M 芯片 npm run build-mac-arm64
Intel芯片 npm run build-mac-x64
通用架构 npm run build-mac
ps；打包好的文件在dist这个文件夹下面

有时候会因为没有签名打包失败，这个时候需要通过签名
# 跳过代码签名（开发阶段）
export CSC_IDENTITY_AUTO_DISCOVERY=false

# 然后打包
npm run build-mac-arm64

Windows 打包
第一步
# 进入项目目录
cd D:\BaiduNetdiskDownload\wps\openwrite-main

# 安装依赖（如果还没安装）
npm install
第二步，安装 electron-builder（如果还没安装）
npm install electron-builder --save-dev
第三步，打包
# 打包 Windows 安装程序（exe）
npm run build-win
# 打包 64 位 Windows 安装程序
npm run build-win

# 或者指定 64 位
electron-builder --win --x64

# 或者打包便携版（免安装）
electron-builder --win -- portable
第四步，打开文件夹
# 打开文件夹
start dist

ps；Windows 需要在 cmd 命令窗口执行，在 vcd 软件是没用的。
