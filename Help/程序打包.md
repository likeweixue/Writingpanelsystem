本文档是用Electron打包。
需要安装依赖，200mb 左右，然后下载依赖失败，请使用淘宝镜像
# 设置 Electron 镜像为淘宝源
export ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/"
第一步，参加package.json文件
"name": "writing-assistant",
  "version": "1.0.0",
  "description": "写作帮手 - 免费的写作软件",
  "main": "main.js",
  "scripts": {
    "start": "electron .",
    "build": "electron-builder",
    "build-mac": "electron-builder --mac",
    "build-mac-arm64": "electron-builder --mac --arm64",
    "build-mac-x64": "electron-builder --mac --x64",
    "build-win": "electron-builder --win"
  },
  "author": "麻昌生",
  "license": "MIT",
  "devDependencies": {
    "electron": "^28.0.0",
    "electron-builder": "^24.6.4"
  },
  "build": {
    "appId": "com.writingassistant.app",
    "productName": "写作帮手",
    "directories": {
      "output": "dist"
    },
    "mac": {
      "target": ["dmg", "zip"],
      "icon": "icon.icns",
      "category": "public.app-category.productivity",
      "hardenedRuntime": true,
      "gatekeeperAssess": false
    },
    "win": {
      "target": "nsis",
      "icon": "icon.ico"
    },
    "files": [
      "**/*",
      "!node_modules/.cache",
      "!dist",
      "!icon.ico"
    ]
  }
}
第二步，准备图标，根目录已经准备了，mac是icon.icns，Windows 是ico.icns
第三步，安装 electron-builder（如果还没安装）
npm install electron-builder --save-dev
第四步，打包架构；
M 芯片 npm run build-mac-arm64
Intel芯片 npm run build-mac-x64
通用架构 npm run build-mac
ps；打包好的文件在dist这个文件夹下面
