主要是删除了一些我感觉不是很好看的诶墨迹图标，显示的是文字内容，然后整体的 UI 配色修改，比如在绿色的护眼主题里面，配色基板上是绿色（冷色）系的。然后是一些小改动，比如标签页面我加高了一点点，这种强迫症看起来好一些些，原来的版本是上面比下面宽。

我好像遇到瓶颈了，我在加一些我不知道长什么样的功能（时间线），然后出现了一些问题，ai 解决不了。

现在的版本的 UI 也有问题（抱歉我实在是不知道怎么改了），理论上建议大家使用绿色主题（后期我会加入自定义功能，比如你可以选择一张照片当软件的背景）。

使用建议，我建议大家尝鲜就行了，不建议用在正式的写作，现在的软件还是测试版，有风险，而且，不知道是不是因为 vpn 的原因，我下载不了npm，我现在打包不了软件，软件只能用浏览器打开 HTML 文件。
