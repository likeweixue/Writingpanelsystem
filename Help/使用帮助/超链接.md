📖 超链接系统使用说明

一、什么是超链接？

超链接允许你在章节之间建立跳转连接，让读者可以快速跳转到相关章节、查看词典词条，或者在文中做标记（锚点）。

格式：[[链接目标]]

示例：

[[第一章]] → 跳转到"第一章"
[[筑基期]] → 打开词典查看"筑基期"
[[#伏笔1]] → 跳转到标记为"伏笔1"的位置
二、三种链接类型

类型	格式	颜色	点击效果
📖 章节链接	[[章节名]]	蓝色 🔵	跳转到该章节
📚 词典链接	[[词条名]]	棕色 🟤	打开词典并选中词条
📍 锚点链接	[[#锚点名]]	橙色 🟠	滚动到锚点位置并高亮
三、如何使用

方法一：手动输入（推荐）

在编辑器中直接输入 [[目标名称]]：

text
主角李三思在[[第一章]]中登场，他修炼的是[[御剑境]]功法。
关于这个功法的详细设定，可以参考[[#功法设定]]。
保存后，[[第一章]]、[[御剑境]]、[[#功法设定]] 会自动变成可点击的链接。

方法二：使用工具栏按钮
选中要作为链接的文字
点击工具栏的 🔗 链接 按钮
选择链接类型（章节/词典/锚点）
选择目标
点击"插入"
四、锚点使用指南

锚点就像书签，可以在文中标记重要位置，方便后续跳转。

1. 添加锚点

方法一：使用工具栏按钮

把光标放在要标记的位置
点击工具栏的 📍 锚点 按钮
输入锚点名称（如：功法设定、伏笔1、重要转折）
方法二：手动输入

在要标记的位置输入：[[#锚点名称]]，然后保存。系统会自动识别。

2. 跳转到锚点

在其他位置输入 [[#锚点名称]]，点击即可跳转。

示例：

text
第一章末尾：
...这就是主角获得的第一件宝物。📍 功法设定 [[#功法设定]]

第十章中：
主角使用的正是[[#功法设定]]中提到的御剑境功法。
点击 [[#功法设定]]，页面会滚动到"📍 功法设定"的位置并高亮显示。

五、常见用途

用途	示例	说明
章节跳转	[[第三章]]	快速跳转到其他章节
伏笔呼应	[[#伏笔1]]	在揭晓伏笔时跳回埋设位置
词典参考	[[御剑境]]	查看术语详细解释
人物关联	[[李三思]]	查看人物设定
设定查阅	[[世界观]]	查看世界观设定
六、注意事项

书名/词条名必须完全匹配
[[第一章]] 和 [[第一章节]] 是不同的链接
词典词条名必须完全一致
锚点名称在同一本书中唯一
如果多个锚点同名，点击会跳转到第一个
链接仅在书籍内部有效
不能跨书籍链接
链接会随内容自动更新
如果章节名或词条名发生变化，链接需要手动更新
七、快捷键

快捷键	功能
Ctrl+Shift+L	插入链接（打开链接对话框）
Ctrl+Shift+A	添加锚点
八、显示效果

在编辑器中，链接会显示为：

章节链接：蓝色虚线下划线 🔵 📖
词典链接：棕色虚线下划线 🟤 📚
锚点链接：橙色虚线下划线 🟠 📍
锚点标记：黄色背景带左侧橙色竖条 📍
鼠标悬停会显示提示信息，点击即可跳转。
