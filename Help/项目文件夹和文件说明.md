项目是 css，HTML，js 的架构，然后根目录下面有几个文件（夹）我解释一下下。
css，这个文件夹苦命有一个 style.css，这个是相当于大的主题文件。
themes，这个文件夹里面有dack.css，default.css，eye.css，oepn.css，warm.css这些 css 对于的是软件的预设主题色（白色，绿色，黄色，暗色，毛玻璃）。
.DS_Store这个文件是相当于遗产，没有什么用处，删除也问题不大，因为我是用 mac 写代码的，理论上每一个 mac 文件夹里面都有这个文件，在 mac 系统看不见，Windows 之类的系统可见。
打包文件，package.json和package-lock.json是那打包的必备文件夹，比如里面有软件的版本，开发者啥的～。
