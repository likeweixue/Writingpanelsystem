# 写作帮手

> 一款开源、免费、自由的写作软件，无需登录，打开即写。

## 特性

- **无需登录**：打开即用，零门槛开始写作
- **功能齐全**：包含常见写作工具——起名、大纲、时间线、角色、设定、校对、取名、地图、查找替换……
- **高度自定义**：可将喜欢的图片设为背景，自由调整文字大小、颜色、字体
- **数据本地存储**：所有数据保存在本地，隐私安全，无需联网
- **完全开源免费**：没有任何隐藏收费

## 适用人群

- 小说创作者
- 剧本写作者
- 内容策划者
- 任何热爱写作的人

## 食用方法
下载一整个源码 https://github.com/likeweixue/word/archive/refs/heads/main.zip
解压之后用浏览器打开index.html（或者把index.html拖到浏览器窗口）就可以体验了

## 特别说明
截止到目前，软件还是测试版，很不稳定，我只建议尝鲜，很不建议用于写作！
## 掏心窝子的话
就是这个项目，并不是我要大赚一笔，我也不靠这个赚钱，我只能按照我自己的想法（写作软件应该是什么样的？）来写的，不过我还是建议大家用那种可以登录的写作软件，比如作家助手，这种有云端备份，安全一点点，也很方便，手机，电脑，ipad 都可以同时写。当然哈，我也不会和国内的文化一样，只会搞“田忌赛马”，那自己的长处去比别人的短处来显优越感，然后发布会各种对比，各种遥遥领先……写作软件就是一个工具而已，写作帮手只是给大家多一个选择而已。
